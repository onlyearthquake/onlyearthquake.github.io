基本语法

类型，Main，语句等较简单过此处略过

编译与执行 C#并不直接编译为机器语言，而是需要CIL的中间托管（可增强可移植性，回收效率，安全性）

使用框架

常用：.NETCORE //.NET Framework //Unity

数据类型
注意事项补充

面向对象

  • 属性{get;set}只读：不加set或=>

  • 方法 可在形参中直接赋值作为默认值（放在最后

  • 构造与析构
    创建对象时运行构造，结束使用析构释放内存

    1. 共同要求：与类的名称相同，无返回值，默认无参数

    2. 不同： 一般函数有默认构造，自己写会覆盖

      析构函数前有~（不构造，无析构

console

6.0支持加入,省去Console

输入

  • .Write不换行
  • .WriteLine换行，可加入格式
  • .ReadKey获取下一个字符或功能键并显示于控制台

输出

  • .Read单个字符
  • .ReadLine一行数据，可换行读取
  • .Beep播放提示音

.Clear清除提示信息

ConsoleKeyInfo

描述按下的控制台键

属性 Key获取控制台键

KeyChar控制台键表示的Uni字符

Modifiers组合键

方法 Equals（）获取值并判断是否等于当前对象

GetHashCode返回哈希值
