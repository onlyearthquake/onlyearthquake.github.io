Basic

基本类型：

num，boolean

string

  • 可以用两个[[来包括一块字符串（如果内容中含有[[，使用[===[）

  • 字符函数(sub,format,char,byte)

table

（关联数组，可视为一个Map）

  • function
  • thread
  • userdata
  • nil

操作符

  • ..连接两个string

  • #str获取字符串的长度

  • —注释内容
  • and or not

变量：

  • 默认为全局变量，除非local修饰

  • 可以同时进行元组的赋值等操作

循环

  • while循环
  • for循环

New Feature In 5.3

  • ==不要使用bit32库==

  • \\整除符号，不会有小数点
